## 🛠️ How to Play

1. **Clone this repository**: 
   ```bash
   git clone https://github.com/YoustinaMitsho/Unity-Projects.git
   ```
   
2. **Navigate to the game folder**:
   ```bash
   cd Unity-Projects/Game-Name
   ```

3. **Extract .rar file**.
   
4. **Open .exe file and enjoy**.
